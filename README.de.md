# genie-playground

Ein Spielplatz zum Experimentieren mit Codegenerierung und KI-Fähigkeiten.

## Über

Dieses Repository dient als Testgelände für verschiedene Codegenerierungstechniken und KI-gestützte Entwicklungsfunktionen.
